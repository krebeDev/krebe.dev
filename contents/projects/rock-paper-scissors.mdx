## About this project
This project is my solution to a [Frontend Mentor](https://www.frontendmentor.io/challenges/rock-paper-scissors-game-pTgwgvgH) challenge.

It is built with ReactJs.

<br />
The designs and style guide were provided. The task was to build the project to meet the following requirements.

<br />

**A user should be able to:**
- View the optimal layout for the game depending on their device's screen size
- Play Rock, Paper, Scissors against the computer
- Maintain the state of the score after refreshing the browser (optional)
- Bonus: Play Rock, Paper, Scissors, Lizard, Spock against the computer (optional)
